# 키의 종류
  
  ### 후보키 
  * 유일성(고유값), 최소성(1개의 속성으로 구성, 복합x) 
   > 예) 학번, 주민등록번호 <br>

  ### 슈퍼키 
  * 유일성유지, 최소성 관련 없음
   > 예)  
       산곡고 3학년 1반 1번 <br>
       산곡고 3학년 1반 31번 <br>
       
  ### 기본키 
  * 튜플을 대표하도록 선정된 후보키
   > 예) 학번
    특징) null 값은 인정 안됨 <br>
  
  ### 대체키 
  >  예) 주민등록번호 <br>
        
  ### 5) 외래키
  * 특정 릴레이션의 기본키를 참조하는 속성 집합
